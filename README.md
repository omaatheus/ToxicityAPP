# ToxicityAPP

O app utiliza a biblioteca TensorFlow para analisar comentários de usuários e determinar se eles são ofensivos ou não.

## Funcionalidades

- **O Que o App faz?**
  - Análise de Sentimento;
  - Detecção de Ofensas;
  - Classificação Final;
  - Interface Simples.
